# QStory - Научные Личности и Их Наследие

Мобильное приложение для изучения истории выдающихся учёных и научных деятелей через интерактивное сканирование QR-кодов и навигацию по карте.

## 🎯 Цель Проекта

Создание удобного интерактивного приложения для получения информации о местах, связанных с выдающимися учёными, с прямым взаимодействием с реальными объектами через QR-коды.

## 📱 Концепт Приложения

QStory - это мобильное приложение, которое позволяет пользователям:
- Сканировать QR-коды на памятных местах и объектах
- Получать детальную информацию о выдающихся учёных и их вкладе в науку
- Просматривать интерактивную карту с метками научных достопримечательностей
- Читать отзывы других пользователей
- Исследовать историю российской науки и техники

## 🛠️ Функциональность

### Основные Возможности
- **QR-Сканирование** - быстрое получение информации о месте и его историческом значении
- **Интерактивная Карта** - навигация по меткам с координатами (широта, долгота)
- **Детальная Информация** - дизайнерски оформленные статьи о местах и личностях
- **Система Отзывов** - возможность оставлять и читать отзывы других пользователей
- **Фотогалерея** - изображения мест и достопримечательностей
- **Ссылочные Материалы** - статьи и научные источники для углубленного изучения

### Технические Особенности
- Адаптивный дизайн для различных устройств
- Интеграция с backend для динамического обновления контента
- Офлайн-доступ к основной информации
- Мультиязычная поддержка
